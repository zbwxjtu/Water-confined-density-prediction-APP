# Water-confined-density-prediction-APP
This app is a MATLAB app where researchers can predict the confined density of water in carbon nanotubes (CNT)

Researchers can install this app with MATLAB software

Input parameters and units: temperature (K), pressure (MPa), bulk density of water (Kg/m^3), diameter of CNT (Å)

Output parameters and units: confined density of water (Kg/m^3)

The best prediction range of input parameters: temperature (600-1173 K), pressure (20-30 MPa), diameter (9.49-50.17 Å), bulk density of water (37.77-699.47 Kg/m^3)
